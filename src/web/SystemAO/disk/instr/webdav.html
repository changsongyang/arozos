Login Method for MacOS

If you are using MacOS, you can select "Connect to Network Server", enter the endpoint of the arozos WebDAV service (//:/webdav/{vroot_name}) and login with your arozos username and password.

Login Method for Windows (Non SSL / TLS Mode)

-- Deprecated since Windows 10 --
To use WebDAV on Windows, you will need a certificate installed or with a reverse proxy with TLS/SSL support

  1. Use a reverse proxy server like Zoraxy to proxy WebDAV traffic with TLS enabled
  2. Click "Map Network Drive" in File Explorer > This PC toolbar
  3. Enter //:/webdav/{vroot_name} as the target
  4. Login with your ArozOS username and password

Login Method for Windows (With SSL / TLS MODE)

If you are using Windows File Explorer as your WebDAV Client, go to "This PC" and add a network drive with the following endpoint: //:/webdav/{vroot_name}
